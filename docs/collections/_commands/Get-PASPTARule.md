# Get-PASPTARule

## SYNOPSIS
Returns risky activities rules from PTA

## SYNTAX

```
Get-PASPTARule [<CommonParameters>]
```

## DESCRIPTION
Returns risky activities rules configured in PTA

## EXAMPLES

### EXAMPLE 1
```
Get-PASPTARule
```

Returns all risky activities rules from PTA
