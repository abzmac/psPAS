# Get-PASPSMServer

## SYNOPSIS
Lists configured PSM Servers

## SYNTAX

```
Get-PASPSMServer [<CommonParameters>]
```

## DESCRIPTION
Allows Vault admins to get a list of all PSM servers defined for an environment.

## EXAMPLES

### EXAMPLE 1
```
Get-PASPSMServer
```

Lists all configured PSM Servers
