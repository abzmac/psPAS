# Get-PASPlatformSafe

## SYNOPSIS
Get safes by platform id

## SYNTAX

```
Get-PASPlatformSafe [-PlatformID] <String> [<CommonParameters>]
```

## DESCRIPTION
Returns all safes for a given platform ID

## EXAMPLES

### EXAMPLE 1
```
Get-PASPlatformSafe -PlatformID WINDOMAIN
```

## PARAMETERS

### -PlatformID
The unique ID/Name of the platform.
